About This System

This application is a prototype for the Assessment Services Branch's Content Inventory Management System. It aims to demonstrate the functional user interface for creating, managing, and tracking Content Inventory records, supporting retention/disposition planning, and functional classification alignment (FCT).

The prototype is built using Vue.js for interactivity and Tailwind CSS for styling, deployed as static HTML pages for client review.
